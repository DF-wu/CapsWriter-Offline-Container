# Web Console documentation

This compatibility entry is kept for existing links. Use the paired guide for
your language:

- [English Web Console client guide](en/web-console.md)
- [繁體中文 Web Console Client 指南](zh-TW/web-console.md)

The Web Console is a client. It serves static UI on port `8080`, while the
browser sends audio to the ASR server's opt-in HTTP API on port `6017`. It does
not contain or run a recognition model.
